Fit2Form: 3D Generative Model for Robot Gripper Form Design

Huy Ha, Shubham Agrawal, Shuran Song

Columbia University

Main Work

  • Design a system that can learn to generate customized gripper fingers for different grasp objects and design objectives.

[!tip] Difference
Design a system that can learn to generate customized object for better grasp(Almost for all kinds of Gripper )?

Grippers

[!tip] Grippers Design
 The majority of robot grippers in industrial applications are highly customized to improve the system’s robustness and accuracy for the target task

Overview

![rhUN37|1280](http://fastly.jsdelivr.net/gh/mingzailao/Pic@master/uPic/rhUN37.png)

[!tip] Workflow

  • Input: 3D object TSDF
  • Generator Network: Generates fingers that maximizes fitness network score.
  • Fitness Network: Given two Fingers and object, output the score.
  • Finally, After Training Generator network, Giving an object, we can get the best fingers.

[!tip] Process of Training

  1. Pretrain of Fitness network
  2. Pretrain of Generator Embedding network
  3. Co-Training of Generator and Fitness network
